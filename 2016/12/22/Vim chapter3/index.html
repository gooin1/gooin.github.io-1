Vim 移动命令

命令 光标动作
j 向下移动一个实际行
gj 向下移动一个屏幕行
k 向上移动一个实际行
gk 向上移动一个屏幕行
0 移动到实际行的行首
g0 移动到屏幕行的行首
^ 移动到实际行的第一个非空白字符
g^ 移动到屏幕行的第一个非空白字符
$ 移动到实际行的行尾
g$ 移动到屏幕行的行尾

命令 j、k、0、$ 都用来操作实际行，如果在之前加上 g 的话，就让 Vim 对屏幕行进行操作。

####　基于单词的移动
Vim 提供了一组动作命令，让我们每次可以把光标正向或反向移动一个单词的距离。
| 命令 | 光标动作 |
| :–: | —————– |
| w | 正向移动到下一单词开头 |
| b | 反向移动到 当前/上一 单词的开头 |
| e | 正向移动到 当前/下一 单词的结尾 |
| ge | 反向移动到上一单词的词尾 |

基于上面的命令， 可以按下 ea 来“在当单词的词尾添加”。另外，也可以把 gea 命令当成“在上一单词词尾添加”的命令。

对字符的查找

Vim 的字符查找命令可以让我们在行内快速移动

f{char} 命令是在 Vim 中移动的最快方式之一。他会在光标位置与当前行行尾间查找指定字符，如果找到了，就把光标跳转到这个字符上；如果没找到，则保持光标不动。

找到字符之后：

  • 按下 ; 跳转到找到的下一个字符，如果按; 太hi按过头了，还可以按 , 返回到上一个字符。

通过查找进行移动

虽然字符查找命令很方便快捷，但是它有个局限就是只能查找当前行的一个字符，要是想查找一个以上的字符或者多行查找的话，就需要用查找命令。

  • /words 可以查找当前光标之后名为 words 的字符（ 表示按下 Enter 键）
  • ?words 可以查找当前光标之前名为 words 的字符
